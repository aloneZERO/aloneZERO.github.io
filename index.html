软件收藏清单

发表于 2017-12-29

整理一下自己觉得好用的软件，避免再次发生想不起名字，狂搜半天的惨痛经历。

日常

  • 浏览器

    • Google Chrome
    • 360 安全浏览器
    • Tor Browser：Access to dark net

  • 安全 / 管理

    • 火绒
    • CCleaner：世界上最受欢迎的 PC 清理和优化工具

  • 音乐

    • 网易云音乐

  • 视频

    • Potplayer
    • oCam：小巧强大的录屏软件

  • 文档

    • 福昕阅读器：PDF
    • Sumatra PDF：开源
    • 极速PDF阅读器
    • Calibre：电子书管理
    • WPS：办公套装

  • 工具

    • Snipaste：不止是截图
    • FastStone Screen Capture：前端开发必备
    • Total Commander：碉堡的文件管理软件
    • 360 压缩
    • 360 驱动大师
    • Everything：文件查找神器
    • Wox：快速启动
    • TeamViewer
    • 猎豹 WiFi
    • 格式工厂
    • Simple Mind：简洁的思维导图
    • KMSAuto-Net：微软软件激活工具（主要是 Office）
    • 1Checker：英文校对

Coding

  • IDE

    • Eclipse
    • Anaconda
    • Jetbrains 全家桶（在线注册地址）
    • RStudio

  • 版本控制

    • Git

  • 科学计算

    • Octave

  • Database GUI Tool

    • SQLyog：MySQL
    • MongoBooster：MongoDB

    • Notepad++：（插件列表）
      • Plugin Manager
      • Compare
      • HEX-Editor
      • HTML Tag
      • JSON Viewer
      • NppFTP

    • EmEditor：超大文本编辑，据说上限 248GB
    • Sublime Text：（插件列表）
      • Package Control
      • Anaconda：ST 华丽变身为 Python IDE
      • Autoprefixer：CSS3 兼容前缀自动补全
      • Emmet
      • HTML-CSS-JS Prettify
      • jQuery
      • Markdown Preview
      • Pretty JSON
      • requirementstxt：Py requirements 文件高亮
      • SideBarEnhancements：ST sidebar 终极增强

    • Microsoft Visio：首选！
    • Gliffy Diagrams：Chrome plugin，可独立运行
    • DIA Diagram Editor
    • Edraw Max：图略丑

    • Cmder：Better cmd
    • Xshell
    • LogViewer：大文本查看，秒开
    • Shadowsocks：SSR作者不在更新，SS仍在

    • Cygwin
    • Worktile：协作平台
    • WAMP
